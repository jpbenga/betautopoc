# Annuaire championnats et médias sportifs

Ce document associe des médias sportifs de référence aux ligues API-Football utilisées ou ciblées par BetAuto.

Règles d'usage:

- Les `ID Ligue` sont des identifiants API-Football.
- Les `ID Média` sont des identifiants internes de documentation BetAuto.
- Cette liste sert de base documentaire pour une future couche de signaux qualitatifs; elle ne doit pas déclencher automatiquement de recherche web ni modifier le prompt d'analyse sans décision explicite.
- Toute information issue d'un média devra être sourcée, datée et distinguée des données API-Football.
